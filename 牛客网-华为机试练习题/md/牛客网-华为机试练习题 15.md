#### 牛客网-华为机试练习题 15

#### 题目描述

输入一个int型的正整数，计算出该int型数据在内存中存储时1的个数。

#### 输入描述:

```
 输入一个整数（int类型）
```

#### 输出描述:

```
 这个数转换成2进制后，输出1的个数

示例1

输入

5

 输出

2
```

#### 解决代码：

```java
import java.util.Scanner;

public class Main{
    public static void main(String[] args){
        Scanner sc  = new Scanner(System.in);
        int num = sc.nextInt();
        int count =0;
        while(num>0){
            num = num& num-1;
            count++;
        }
        System.out.println(count);
    }
}
```
