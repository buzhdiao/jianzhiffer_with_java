#### 腾讯50题-231_2的幂

给定一个整数，编写一个函数来判断它是否是2的幂次方

```
输入：1
输出：true
解释：2^0=1
```

#### 思路

判断:

* 如果n=0，返回false

* 如果n=1，返回true

* 如果n/2==0，那就递归调用求解算法
* 否则，返回false

#### 解决代码：

```java
class Solution {
   public static boolean isPowerOfTwo(int n){
		if(n == 0)
			return false;
		if(n == 1)
			return true;
		
		if(n % 2 == 0){
			return isPowerOfTwo(n/2);
		}else
			return false;
	}
}
```
